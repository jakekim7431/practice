
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the generated numbers to the user.
*   Modern, clean user interface.

## Implemented Features

### Version 1.0

*   Basic HTML structure.
*   Button to generate numbers.
*   JavaScript to generate and display numbers.
*   Basic styling for the page.
*   Use of a web component to encapsulate the lotto generator functionality.
